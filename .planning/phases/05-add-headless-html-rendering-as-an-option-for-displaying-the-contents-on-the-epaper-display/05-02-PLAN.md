# Plan 05-02: wkhtmltoimage Backend (Full HTML/CSS)

## Goal
Add `wkhtmltoimage` as an optional second backend for `Papyrus.Renderer.Headless` — enabling full HTML/CSS rendering for deployments that can accommodate the ~100MB binary.

## Context
This is the **second backend** in the hybrid rendering approach. It targets deployments where full CSS/HTML support is needed and the wkhtmltoimage binary can be installed (servers, larger embedded systems with storage).

**Tradeoffs:**
- ✅ Full HTML5/CSS3 support (flexbox, grid, web fonts)
- ✅ Mature, stable technology (archived but production-ready)
- ✅ No network dependency (fully offline)
- ❌ ~100MB binary + Qt dependencies
- ❌ Archived project (no new features, security updates)
- ❌ ARM builds available but not actively maintained

## Requirements Addressed
- **RENDER-04**: HTML to bitmap rendering — full CSS backend path

## Pre-conditions
- Plan 05-01 complete: SVG backend works via `resvg`
- `Papyrus.Bitmap.from_image/2` exists and works
- `Papyrus.Display.display/2` exists and works

### Task 2: Add wkhtmltoimage backend to Headless module

**Files to create/modify:**
- `lib/papyrus/renderer/headless.ex` — add `:wkhtmltoimage` backend support

**Implementation details:**

Extend `Papyrus.Renderer.Headless` with backend selection:

1. **Update `render_html/3`** (now 3 args with opts):
   ```elixir
   def render_html(input, spec, opts \\ [])

   opts:
     :backend — :auto (default), :svg, :wkhtmltoimage
     :wkhtml_path — path to wkhtmltoimage binary (default: System.find_executable)
   ```

2. **Backend selection logic:**
   ```elixir
   defp select_backend(:auto) do
     cond do
       wkhtmltoimage_available?() -> :wkhtmltoimage
       Code.ensure_loaded?(Resvg) -> :svg
       true -> {:error, "no rendering backend available"}
     end
   end

   defp select_backend(:svg), do: :svg
   defp select_backend(:wkhtmltoimage) do
     if wkhtmltoimage_available?(), do: :wkhtmltoimage,
       else: {:error, "wkhtmltoimage not found in PATH"}
   end
   ```

3. **New private function: `render_with_wkhtml/2`:**
   ```elixir
   defp render_with_wkhtmltoimage({:html, html}, spec, opts) do
     path = Keyword.get(opts, :wkhtml_path, "wkhtmltoimage")

     # Build command args
     args = [
       "--width", to_string(spec.width),
       "--height", to_string(spec.height),
       "--disable-smart-width",
       "--javascript-delay", "0",
       "--quiet",
       "-", "-"  # stdin input, stdout output
     ]

     # Run port
     opts = [:binary, :exit_status, :stderr_to_stdout, :use_stdio]
     case System.cmd(path, args, opts ++ [stdin_data: html]) do
       {png_binary, 0} ->
         # Pass to Bitmap.from_image/2
         Papyrus.Bitmap.from_image(png_binary, spec)

       {_, exit_code} ->
         {:error, "wkhtmltoimage exited with code #{exit_code}"}

       {:error, reason} ->
         {:error, "wkhtmltoimage failed: #{reason}"}
     end
   end
   ```

4. **Input handling for wkhtmltoimage:**
   - `{:html, html}` — pass via stdin
   - `{:url, url}` — pass URL as argument (not stdin)
   - `{:file, path}` — read file, pass as HTML, or pass path directly

5. **`wkhtmltoimage_available?/0` helper:**
   ```elixir
   defp wkhtmltoimage_available? do
     System.find_executable("wkhtmltoimage") != nil
   end
   ```

**Acceptance criteria:**
- [ ] `render_html/3` accepts `:backend` option
- [ ] `:auto` backend tries wkhtmltoimage, falls back to resvg
- [ ] `:wkhtmltoimage` backend returns error if binary not found
- [ ] `:svg` backend forces resvg usage
- [ ] URL input works with wkhtmltoimage backend

## Verification
```bash
# Test SVG backend (always works):
mix run -e """
  alias Papyrus.Renderer.Headless
  spec = %Papyrus.DisplaySpec{model: :test, width: 800, height: 480, buffer_size: 48000}
  html = Papyrus.TestPattern.Html.simple_message("Test")
  {:ok, bitmap} = Headless.render_html(html, spec, backend: :svg)
  IO.puts("SVG: #{byte_size(bitmap)} bytes")
"""

# Test wkhtmltoimage backend (if installed):
mix run -e """
  alias Papyrus.Renderer.Headless
  spec = %Papyrus.DisplaySpec{model: :test, width: 800, height: 480, buffer_size: 48000}
  html = Papyrus.TestPattern.Html.simple_message("Test")
  case Headless.render_html(html, spec, backend: :wkhtmltoimage) do
    {:ok, bitmap} -> IO.puts("wkhtml: #{byte_size(bitmap)} bytes")
    {:error, e} -> IO.puts("wkhtml not available: #{e}")
  end
"""

# Test auto-selection:
mix run -e """
  alias Papyrus.Renderer.Headless
  spec = %Papyrus.DisplaySpec{model: :test, width: 800, height: 480, buffer_size: 48000}
  html = Papyrus.TestPattern.Html.simple_message("Test")
  {:ok, bitmap} = Headless.render_html(html, spec)  # :auto backend
  IO.puts("Auto backend: #{byte_size(bitmap)} bytes")
"""
```
